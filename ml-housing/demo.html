House Price Prediction

A supervised linear-regression model trained on 2,000 homes to predict sale price from six features. This page runs the real trained model — the exact learned weights — right in your browser. Drag any feature and the prediction updates live. Test-set accuracy: R² = 0.94.

Each dot is a test home the model never saw during training. The dashed line is a perfect prediction — the tighter the cloud hugs it, the better the fit. Here it explains 94% of the variance in price.

How it works: features are standardized, then a linear regression is fit two ways — gradient descent (SGDRegressor) and the closed-form normal equation (LinearRegression) — and evaluated on a held-out 20% test set. This demo uses the trained weights directly, so every prediction above is the genuine model output, not an approximation. See the source and write-up.

How it worksBehind the demo
01

Ordinary least squares

Features are standardized, then the coefficients are solved in closed form via the normal equation — the same weights this page ships and predicts with.

02

Held-out evaluation

1,600 homes train the model, 400 are kept back. Every metric above — R², RMSE, MAE — is measured on homes the model never saw.

03

Comparable importance

Raw coefficients live in different units, so they're rescaled by each feature's spread. That's what makes the influence bars honest against each other.
