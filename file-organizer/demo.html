File Organizer

A Python CLI that sorts a messy folder into subfolders by file type. This demo runs the real algorithm in your browser — the same categorization, --dry-run preview, collision-safe renaming, and undo. The terminal on the right mirrors the actual script's output.

Try this: flip on Dry run and hit Organize — nothing moves, you just see the plan. Turn it off to sort for real, then Add new downloads (which drops in a file whose name is already taken) and Organize again to watch collision-safe renaming (vacation_1.jpg) in action. Undo reverses the last real run using its logged moves — exactly like the CLI.
